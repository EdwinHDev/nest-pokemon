# Levantar proyecto en desarrollo

1. Clonar el repositorio
```
url
```
2. Ejecutar comandos
```
npm install
```
3. Tener Nest CLI instalado (Ejecutar como administrador)
```
npm i -g @nestjs/cli
```
4. Instalar base de datos (Mongodb)
  * Tener docker desktop instalado
```
docker compose up -d
```

## Stack
* Nestjs
* Mongodb
